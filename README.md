# Image Steganography Comparative Study

## Overview
This project implements and compares multiple image steganography techniques:
- LSB (Least Significant Bit)
- PVD (Pixel Value Differencing)
- DCT-based method
- AES encryption for security

## Features
- Encode secret data into images
- Decode hidden messages
- Performance evaluation (PSNR, SSIM, MSE)
- GUI-based application

## How to Run
1. Install dependencies
2. Run:
   python modern_app.py

## Dataset
Sample images are included for testing.

## Research Contribution
This repository supports the experimental evaluation presented in the research paper.
